# Especificaciones Funcionales por Módulo

Cada módulo de AGROERP se documenta copiando la plantilla oficial:

**Plantilla:** [`../FUNCTIONAL_SPECIFICATION_TEMPLATE.md`](../FUNCTIONAL_SPECIFICATION_TEMPLATE.md)

## Convención de nombres

```
{CODIGO}_FUNCTIONAL_SPEC.md
```

Ejemplos:
- `PRM_FUNCTIONAL_SPEC.md`
- `CPE_FUNCTIONAL_SPEC.md`
- `EITE_FUNCTIONAL_SPEC.md`
- `QMCL_FUNCTIONAL_SPEC.md`
- `FMTP_FUNCTIONAL_SPEC.md`
- `PSMP_FUNCTIONAL_SPEC.md`
